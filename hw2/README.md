## 作業二－電商前端網頁
#功能
1. 主畫面含「查詢和新增」兩個按鈕
2. 查詢鈕按下會出現往下滑的圖形動畫，往下滑後會看到商品和頁碼
3. 按下新增紐會引導到新分頁，該分頁可新增商品資訊
4. 新增商品之頁面有「增加」、「清除」、「回到主畫面」等按鈕
    * 「增加」－將輸入的資料加入資料庫
    * 「清除」-將頁面中已輸入的資料清除
    * 「回到主畫面」- 回到index.html

#設計
1. 將jumbotron變為滿版，底色為綠色
2. 主畫面搜尋按鈕樣式增加圖案和hover效果
3. 新增頁面之按鈕加上active效果
